CentralAsia (UZ)  -  В понедельник, 28 февраля, в 17:15 по ташкентскому времени из польского города Катовице вылетел специальный самолет авиакомпании Uzbekistan Airways с гражданами Узбекистана, эвакуированными из Украины. Об этом сообщил официальный представитель Министерства иностранных дел Узбекистана.
Среди эвакуируемых — женщины, дети, студенты и работники различных сфер.
В настоящее время на КПП «Карчова» (Польша) организован накопительный пункт для граждан Узбекистана, прибывающих из украинского Львова. В течение дня 28 февраля будет организован дополнительный спецборт для вывоза узбекистанцев.
Второй борт уже вылетел из Ташкента. Его прибытие в Польшу ожидается около 21:00 по ташкентскому времени.
По информации посольства Узбекистана в Киеве, для ускорения доставки граждан из Львова до КПП организована аренда семи автобусов. При этом во Львов продолжают прибывать граждане Узбекистана для организованного выезда на родину.
